# Model

### Model.create(values)
Create model from data object.

| Parameter | Type | Description |
| - | - | - |
| `values` | `object` | Data object to map |


### merge(values)
Update model properties.

| Parameter | Type | Description |
| - | - | - |
| `values` | `object` | Data object to map |

### serialize()
Serialize model to plain object.

### clone()
Clone model to new instance.
